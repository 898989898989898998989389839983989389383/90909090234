## Tawk.to Dashboard Setup

### Access Dashboard:
**URL**: https://dashboard.tawk.to/

### Recommended Settings:
1. **Business Hours**: Set your support hours
2. **Offline Form**: Enable for after-hours messages
3. **Pre-chat Form**: Collect user name/email (optional)
4. **Widget Position**: Bottom right (default is good)
5. **Widget Color**: Match your app colors (blue #3b82f6)
6. **Notifications**: Enable email/mobile notifications

### Dashboard Features:
- View all conversations
- See user information
- Chat history
- Analytics (response time, satisfaction)
- Multiple agents support
- Mobile app for agents
